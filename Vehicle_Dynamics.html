0100 km/h Vehicle Acceleration Simulation Using Python with distance and velocity graph

Introduction

In this project, I simulated how long the vehicle takes to accelerate from 0 to 100 km/h using a simple simulation code and logic of physics based numerical mmodel in Python. The goal of this simulation is to understand how the varies forces acts on the vehicle like drag force , rolling resistance force and Engine taractive force. Also to learn the impace of vehicle mass , engine power and traction limits on acceleration.

Assumptions

To simplify the simulation i have assumed the following : 1. The engine provides constant power throughout the acceleration. 2. Gear shifts, turbo lag and drivetrain losses are ignored. 3. The road is flat (no inclination). 4. Rolling resistance is constant. 5. Air density remains constant. 6. aerodynamic drag increases with square of velocity.

Vehicle Parameters

The parameters used in this simulation: vehicle mass(m) = 1200 kg Engine power (P) = 150 kW Drag coefficient (Cd)= 0.29 Frontal Area(A) = 2.2 m^2 Air density() = 1.225 kg/m^2 Rolling Resistance coefficient(Crr) = 0.015 Wheel Radius (r) = 0.3m

Equations Used

1. Aerosynamic Drag Force : F_drag = 0.5 * * Cd * A * v^2 2.Rolling Resistance Force: F_rr = m * g * Crr 3.Engine Tractive Force from Power: F_engine = P / v if v = 0 , the denominator is taken as 1 4. Net force on Vehicle: F_net = F_engine - F_rr - F_drag 5. Acceleration: a = F_net / m 6. Velocity update: new_v = old_v + a *dt

Simulation Logic

1.Starting at v = o m/s 2. Using a small step (dt = 0.01 seconds) 3. In each step : - computing drag force - computing rolling resistance - computing available engine force - computing net force - computing acceleration - Updating velocity with Euler integration 4. Stop when the velocity reaches 100 km/h (27.78 m/s)

Python Code

In[7]:
import numpy as np
import matplotlib.pyplot as plt
In[8]:
mass = 1200      # kg
power = 80_000   # watts (150 kW)
drag_coeff = 0.29
frontal_area = 2.2
air_density = 1.225
rolling_resistance = 0.015
In[9]:
dt = 0.01  # time step
time = [0] #both are list 
velocity = [0] 
acceleration = [0]
distance = [0]
In[10]:
while velocity[-1] <= 27.78:  # 100 km/h in m/s
    v = velocity[-1]

    # Resistive forces
    F_drag = 0.5 * drag_coeff * frontal_area * air_density * v**2 #drag force 
    F_rr = mass * 9.81 * rolling_resistance #rolling resistance 

    # Available driving force from power (F = P / v, limit when v=0)
    F_drive = power / max(v, 1)  #engine power 

    # Net force
    F_net = F_drive - F_drag - F_rr

    # Acceleration
    a = F_net / mass

    # Euler integration
    new_v = v + a * dt
    new_t = time[-1] + dt

    velocity.append(max(new_v, 0)) #the max value between new velocity and 0 is appended to the list
    time.append(new_t)
    acceleration.append(a)
    new_s = distance[-1] + v * dt
    distance.append(new_s)
zero_to_hundred_time = time[-1]
print("0-100 km/h time:", zero_to_hundred_time , "seconds")

Analysis

From the simulation, the vehicle accelerates quickly at the beginning but becomes slower as the speed increases. This happens because: 1. At low speeds: - The engine can give a lot of force. - Air drag is very small. - So the car speeds up quickly. 2. At medium speeds: - Air drag starts increasing. - The engine force becomes a little less effective. 3. Near 100 km/h: - Air drag becomes very strong. - The engine needs more power to push the car forward. - So acceleration becomes slower. That is why the velocity curve rises fast at the start and slowly near the end. This behavior is normal for all real cars.
