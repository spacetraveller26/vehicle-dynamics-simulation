# 0–100 km/h Vehicle Acceleration Simulation (Python)

This project models how a vehicle accelerates from 0 to 100 km/h using simple physics.  
The simulation uses engine power, drag force, rolling resistance, and vehicle mass to compute the acceleration.

## Technologies Used
- Python
- Jupyter Notebook
- Matplotlib

## What I Learned
- How vehicle drag affects acceleration
- Basics of vehicle dynamics (longitudinal)
- How to build physics-based simulations
- How to visualize performance using graphs

## Why I Built This
This project helps me build foundational skills for automotive engineering and vehicle simulation.  
It is also the first step toward more advanced simulations like torque curves, gear-shift models, and full vehicle dynamics.
